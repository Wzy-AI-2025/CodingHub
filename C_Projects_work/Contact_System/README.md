# C语言单链表版通讯录管理系统
纯C语言实现，采用**带头结点单向链表**存储数据，代码完全模块化拆分，支持本地文件持久化读写。

## 项目文件结构
- `contact.h`：头文件，结构体定义、宏定义、所有函数声明
- `contact.c`：全部功能逻辑实现代码（增删改查、文件读写、链表操作）
- `main.c`：程序入口、菜单交互、主循环调度
- `contacts.txt`：自动生成的通讯录数据存储文件

## 核心功能
1. **添加联系人**：头插法新建节点，自动校验姓名唯一性
2. **删除联系人**：通过姓名定位节点并释放内存
3. **查询联系人**：按姓名精准检索信息
4. **修改联系人**：可选修改姓名、性别、电话、地址
5. **展示全部联系人**：遍历链表打印所有数据
6. **一键清空通讯录**：循环释放所有节点内存
7. **持久化存储**
   - 程序启动自动从 `contacts.txt` 加载数据
   - 每次增、删、改、清空操作后自动保存到文件

## 技术细节
- 数据结构：带头结点单向链表
- 内存管理：手动 malloc 创建节点，free 释放删除节点，无内存泄漏
- 插入方式：头插法
- 防重机制：添加时检测同名联系人，禁止重复录入

## 运行方式
直接编译 `main.c` + `contact.c` 运行程序，输入数字选择对应功能，输入 `0` 退出程序。
